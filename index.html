debug.js Easy Debugging w/o F12
Logging, Time Test, Element Inspection, Screen Measure, and more
// Usage

Embed the debugger into your web page
Include the script file. Download 
<script src="debug.js"></script>
That's it!

The debug window will be automatically generated and is draggable.

‚ö†The debug window is hidden by default setting.
You can switch hide/show by F2 key.

Logging
Put log messages anywhere you want.
log(message);
e.g.,)
<!DOCTYPE html>

// Methods
Name Argument Description
log message string Output standard log
log.e message string Output error log
log.w message string Output warning log
log.i message string Output information log
log.d message string Output debug log
log.s message string Output system log
log.p arg1: variable
arg2: message string(opt) 		Output object dump
log.t arg1: timer name
arg2: message string 		Output log w/ elapsed time 
log.t('timer1', 'foo')
-> 00:00:12.345 foo

log.t(null, 'foo')
-> 00:00:12.345 foo
default timer name will be used in case null is specified for the timer name.
log.stack Output stack trace
log.clear Clear log message
time.start arg1: timer name(opt)
arg2: message(opt) 		Start a timer 
time.start()
-> timer0: timer started
default timer name will be used in case no arguments are specified.

time.start('timer1')
-> timer1: timer started

time.start('timer1', 'Timer Start: %n')
-> Timer Start: timer1

time.start('timer1', null)
-> no output log (only start the timer)
time.split arg1: timer name(opt)
arg2: message(opt) 		Print elapsed time of the timer 
time.split()
-> timer0: 00:01:02.345 (‚äø00:01:02.345)
default timer name will be used in case no arguments are specified.

time.split('timer1')
-> timer1: 00:01:02.345 (‚äø00:01:02.345)

time.split('timer1')
-> timer1: 00:01:03.345 (‚äø00:00:01.000)

time.split('timer1', '[%n] epalsed time: %t (%lt)')
-> [timer1] elapsed time: 00:01:03.345 (‚äø00:00:01.000)

time.split('timer1', null)
-> no output log (only split the timer)
time.end arg1: timer name(opt)
arg2: message(opt) 		Print elapsed time of the timer and clear. 
time.end()
-> timer1: 00:01:02.345
default timer name will be used in case no arguments are specified.

time.end('timer1')
-> timer1: 00:01:02.345

time.end('timer1')
-> timer1: 00:01:03.345 (‚äø00:00:01.000)

time.end('timer1', '[%n] epalsed time: %t (%lt)')
-> [timer1] elapsed time: 00:01:03.345 (‚äø00:00:01.000)

time.end('timer1', null)
-> no output log (only clear the timer)
time.check timer name Get elapsed time of the timer. 
time.check()
-> A string like "00:01:23.456" will be returned.
default timer name will be used in case no arguments are specified.

time.check('timer1')
-> A string like "00:01:23.456" will be returned.
